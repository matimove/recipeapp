## FermentationLogBook
This is a CRUD web app project made for the course TKT20019 Tietokannat ja web-ohjelmointi (Databases and web-programming)
The idea is a web application where users can add and browse fermentation projects and comment on them.

# Fermentation Log Book
- In the web application users can add fermentation run details.
- User can create a profile and sign in and out.
- User can add, edit or remove fermentation runs.
- User can see fermentation runs added by other users.
- User can search for fermentation runs using searchwords.
- Profile page shows how many fermentation run the user has added and lists them.
- User can select tags for each fermentation run (Fermentation type, Application purpose and Organism type).
- Users can comment on fermentation runs added by other people.
